This directory holds baseline projects and datasets used by this research repo.

- `agentbench/` is the AGENTBench codebase, tracked as a git submodule.
- `agentbench_hf/` is the AGENTBench Hugging Face dataset repo, tracked as a git submodule.
- `swd-bench/` is the SWD-Bench project snapshot used locally in this repository.
